# AI-Notes-Summarizer
The AI Notes Summarizer is a Natural Language Processing (NLP) project developed using Python, Flask, and Hugging Face Transformers. The application automatically summarizes long notes into shorter meaningful summaries using deep learning models.

# AI Notes Summarizer

## Project Overview

The AI Notes Summarizer automatically summarizes long text into concise summaries using NLP and Transformer models.

The project uses Hugging Face Transformers and Flask.


# Features

- AI-based summarization
- Long note compression
- NLP processing
- Web interface


# Technologies Used

- Python
- Flask
- Transformers
- PyTorch


# Installation

Install required libraries:

pip install flask transformers torch

# Run Project

python app.py


# AI Flow

Input Text → NLP Model → Summary
